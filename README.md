# HealthSites
Kotlin Android App


## Requirements

* Android Studio
* Firebase account (Firebase firestore database)
* Git

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/albeertito7/HealthSites.git
```
